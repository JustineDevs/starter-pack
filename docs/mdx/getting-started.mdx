# Getting Started

Welcome to the Project Planning Starter Pack. This guide will help you get started in 3 minutes.

## Quick Setup

### Step 1: Review the Structure

This repository contains:

- **12 Planning Sections** - Complete strategic and operational planning templates
- **Technical Execution Guide** - Comprehensive implementation guide
- **7 Claude Skills** - Auto-activated AI assistance
- **Reference Guides** - Quick access to technical information

### Step 2: Start with Planning

Begin with the planning sections:

1. Read [Executive Summary](/planning/1-Executive-Summary.md)
2. Define [Objectives & Metrics](/planning/2-Objectives-Success-Metrics.md)
3. Set [Scope Definition](/planning/3-Scope-Definition.md)
4. Design [System Architecture](/planning/4-System-Architecture-Design.md)

### Step 3: Use Claude Skills

Ask Claude about your project:

- "What tech stack should we use?"
- "Create an ADR for our database choice"
- "Review this code for quality"
- "Set up CI/CD pipeline"
