# LLM Content Directory

This directory contains **raw, unprocessed outputs** from Large Language Models.

## Structure

```
content/
├── claude/      # Claude-generated content
├── gpt/         # GPT-generated content
├── gemini/      # Gemini-generated content
└── other/       # Other LLM-generated content
```

## What Goes Here

- Direct copy-paste from LLM conversations
- Initial drafts and rough outputs
- Multiple versions or iterations
- Experimental or exploratory outputs
- Unreviewed content

## File Naming

Use format: `[date]-[llm]-[type]-[description].[ext]`

Example: `2026-01-20-claude-code-api-routes.ts`
