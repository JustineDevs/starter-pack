# LLM Output Directory

This directory contains **finalized, processed outputs** from Large Language Models.

## Structure

```
output/
├── code/        # Generated code files
├── docs/        # Generated documentation
├── plans/       # Generated planning documents
└── analysis/    # Generated analysis and reports
```

## What Goes Here

- Reviewed and approved LLM outputs
- Code that has been validated
- Documentation that's been edited
- Analysis that's been verified
- Ready-to-use artifacts

## File Naming

Use format: `[date]-[llm]-[type]-[description].[ext]`

Example: `2026-01-20-claude-code-api-routes.ts`

## Integration

After review, these outputs can be:
- Integrated into the project (`src/`, `docs/`, etc.)
- Used as reference material
- Shared with team members
- Committed to version control
