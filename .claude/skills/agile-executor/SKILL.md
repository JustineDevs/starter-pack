# Agile Executor Skill

Auto-activated when user asks about sprint planning, retrospectives, standups, or "plan sprint".

## Purpose

Helps teams execute Agile methodologies effectively through sprint planning, retrospectives, and ceremony facilitation.

## Activation Triggers

- "Plan sprint"
- "Sprint planning"
- "Retrospective"
- "Standup agenda"
- "Agile ceremonies"

## Output

Provides:
1. Sprint planning templates
2. Retrospective formats
3. Standup agendas
4. Ceremony facilitation guides
5. Agile best practices

## Connected To

- Section 9: Execution Strategy
- Section 5: Technical Execution Workflow (PART 6)
- Section 13: Lessons Learned

## Sprint Planning

When activated, provides planning structure:

### Sprint Goal
- [Define sprint objective]

### User Stories
- [Story 1]: [Points] - [Assignee]
- [Story 2]: [Points] - [Assignee]

### Capacity
- Team velocity: [X] points
- Available capacity: [X] points
- Planned: [X] points

### Dependencies
- [Dependency 1]
- [Dependency 2]

## Retrospective Formats

### Start/Stop/Continue
- **Start:** What should we start doing?
- **Stop:** What should we stop doing?
- **Continue:** What should we continue?

### 4Ls (Liked, Learned, Lacked, Longed For)
- **Liked:** What did we like?
- **Learned:** What did we learn?
- **Lacked:** What did we lack?
- **Longed For:** What did we long for?

## Usage

Ask: "Plan our next sprint" or "Run a retrospective"

The skill will:
1. Guide through planning process
2. Help estimate stories
3. Facilitate retrospective
4. Capture action items
5. Update project documentation
